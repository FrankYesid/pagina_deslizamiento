    Roadmap de Desarrollo

    Fase 1: Integración de Datos

    Conectar con la API de datos de precipitaciones y desarrollar el pipeline de datos atmosféricos.

    Fase 2: Desarrollo del Modelo

    Entrenamiento del modelo de IA para detectar cambios superficiales basados en datos climáticos.

    Fase 3: Validación y Pruebas

    Validación del modelo con datos históricos y pruebas en campo en la región de Caldas.

    Fase 4: Despliegue y Escalabilidad

    Implementación en producción y expansión a otras regiones cafeteras de Colombia.
